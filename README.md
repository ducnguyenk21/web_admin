# 💪 Fitness Web Admin

Ứng dụng quản trị giúp quản lý toàn bộ hệ thống tập luyện thể thao, bao gồm bài tập, món ăn, nguyên liệu, người dùng và thống kê hoạt động.

## 🚀 Tính năng chính

- ✅ Đăng nhập admin
- ✅ Quản lý bài tập: thêm, sửa, xóa, tìm kiếm bài tập
- ✅ Quản lý nhóm bài tập
- ✅ Quản lý nguyên liệu
- ✅ Quản lý món ăn: thêm công thức, chỉnh sửa calo, nguyên liệu
- ✅ Quản lý người dùng và thông tin tài khoản
- ✅ Quản lý bình luận từ người dùng
- ✅ Thống kê tổng quan: số lượng bài tập, người dùng, món ăn, bình luận,...
- ✅ Giao diện thân thiện – quản lý trực tiếp trên trình duyệt
