# 💪 Fitness App

Ứng dụng hỗ trợ người dùng tập luyện thể thao, theo dõi calo, chế độ ăn và giấc ngủ.

## 🚀 Tính năng chính

- Đăng nhập / Đăng ký
- Lựa chọn bài tập theo mục tiêu
- Tính toán và thống kê calo tiêu thụ
- Gợi ý thực đơn theo bữa
- Theo dõi giấc ngủ và biểu đồ
